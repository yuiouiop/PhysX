NVIDIA APEX 1.4.1 SDK distribution

Note: APEX SDK has been deprecated!

Visual Studio based APEX projects:
----------------------------------
Important Note: Building the APEX SDK with the provided solutions will overwrite the pre-built APEX binaries with versions that don't support GPU acceleration.
There are no pre-built APEX binaries for Visual Studio 2017. The Visual Studio 2017 build doesn't support GPU acceleration.
To run samples or snippets you need to build PhysX first and run the script ./bin/run_dllcopy.cmd
